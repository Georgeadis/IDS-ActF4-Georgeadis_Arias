# Ejecutar proyecto Inventario

Pasos para compilar y ejecutar localmente en Windows (PowerShell o CMD):

PowerShell:
```powershell
cd "d:\Universidad _Ciudadana\POA_ACT4\IDS-ActF4-Georgeadis_Arias\Inventario"
.\run.ps1
```

CMD / Git Bash (Windows):
```cmd
cd "d:\Universidad _Ciudadana\POA_ACT4\IDS-ActF4-Georgeadis_Arias\Inventario"
run.bat
```

Notas:
- Asegúrate de tener `javac` y `java` en el PATH (JDK instalado).
- El script compila los .java en `src` y coloca las clases en `bin`.
- Si ves errores, revisa los mensajes de compilación y corrige los archivos indicados.
